Salut! Prietenii tăi pot deschide acest fișier în Chrome

Acesta este un exemplu de pagină statică HTML + CSS + JavaScript. Salveaz-o ca index.html și trimite fișierul prietenilor — ei doar îl deschid în browser.

1. Informație

Text, imagini și linkuri — toate funcționează dintr-un singur fișier.

2. Formular (local)

Acest formular nu trimite date către server — este doar demonstrativ.

3. JavaScript interactiv

Apasă butonul de mai jos pentru a vedea o acțiune JavaScript.

4. Publicare

Poți urca fișierul pe GitHub Pages sau pe un serviciu de hosting static să fie accesibil online.

Cum să îl trimiți prietenilor

  1. Salvează fișierul ca index.html.
  2. Trimite-l prin e-mail, Telegram, Discord sau Google Drive — prietenii îl descarcă și deschid în Chrome.
  3. Dacă vrei să fie online: urcă fișierul pe GitHub și activează GitHub Pages, sau folosește Netlify / Vercel pentru hosting gratuit.
